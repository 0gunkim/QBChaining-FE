## 1. 팀 소개
- REACT, NODE.JS
- 개발인원 6명
  > FRONT_END:  윤창순 | 김영건

  > BACK_END :  윤태식 | 윤상돈 | 이지훈

  > DESIGNER :  최정빈
  
## 2. 프로젝트 설명
- 개발언어 : JAVASCRIPT | REACT
- 협업 툴 : GIT | NOTION | FIGMA 

- 프로젝트 취지 : 개발프로그램 특성상 영어로된 질문과 답변들이 많은데, 한국 사람들이 한글로된 질문과 답변을 보며 조금 더 빠르게 에러를 해결할 수 있는 플랫폼을 만들고자 함
